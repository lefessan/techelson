Techelson

Techelson is a Testing Framework for Michelson, the smart contract language for Tezos. It is also directly available from contracts written in the Liquidity language.

Example

A simple example of what you can do with Techelson:

{
    PUSH @storage nat 0 ;
    PUSH @amount mutez 3 ;
    PUSH @delegatable bool True ;
    PUSH @spendable bool True ;
    NONE @delegate key_hash ;
    PUSH key "manager address" ;
    SHA512 @manager ;
    PRINT_STACK ;            # print the stack at this point
    STEP "before creation" ; # stop interpretion here

    CREATE_CONTRACT "SimpleExample" ;  # originate the contract
    PRINT_STACK ;            # print the stack again here
    STEP "after creation" ;
}
